# Meters App

Приложение для отображения списка счётчиков горячей и холодной воды

## Дополнение:
Адреса кэшируются и не запрашиваются повторно.

Для загрузки адресов используется ресурс /areas/ и параллельные запросы только по неизвестным id.

Batch-формат со списком id на тестовом стенде не отрабатывал корректно, поэтому выбран рабочий вариант с параллельной загрузкой и кэшированием.

## Что реализовано

- Загрузка списка счётчиков с API `/meters/`
- Пагинация по 20 элементов на страницу через `limit` и `offset`
- Внутренний скролл таблицы
- Фиксированная шапка таблицы
- Отображение:
  - порядкового номера
  - типа счётчика
  - даты установки
  - автоматического/неавтоматического счётчика
  - текущих показаний
  - адреса
  - примечания
- Удаление счётчика через `DELETE /meters/:meterId/`
- После удаления данные страницы загружаются заново
- Кэширование адресов: уже загруженные адреса повторно не запрашиваются
- Параллельная загрузка неизвестных адресов
